# Webfinity 2026 Starter

## Project

This repository is a reusable starter kit for frontend hackathons and design competitions.

Repository name:

`webfinity-2026`

The immediate goal is to prepare for the Webfinity competition.

Competition format:

- Theme announced at the start
- 2.5 hours design & development
- 1.5 hour presentation

The starter should eliminate setup time so only design and implementation remain on competition day.
